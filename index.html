Welcome to Pattern Libary

Pattern libraries are a collection of user interface design elements, with pattern libraries these particular elements are defined in the way that they are coded, how they behave and how they look. Inside this pattern library documentation is provided for each different component is so that developers and designer can understand each component.

Setup

CSS

To include this CSS stylesheet, name it as the first stylesheet within the head of the HTML page.

<link rel="stylesheet" type="text/css" href="PatternLibraryTest.css">

Font Awesome Icons

To include these icons into your website, name it as the second stylesheet within the head of the HTML page.

<link rel="stylesheet" href="https://use.fontawesome.com/releases/v5.8.2/css/all.css">

HTML Structure

Use this based HTML structure to start implementing the Pattern Library

<!DOCTYPE html> <html> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <!-- update font awesome link --> <link rel="stylesheet" type="text/css" href="PatternLibraryTest.css"> <link rel="stylesheet" href="https://use.fontawesome.com/releases/v5.8.2/css/all.css"> </head> <body> </body> </html>
